1d1d动态规划决策单调性

发表于2026-05-14|更新于2026-05-14
|浏览量:

1d1d 动态规划是一类计算如下类型 dp 式子的 dp。

fi=max⁡fj+w(i,j)f_i = \max f_j + w(i, j) fi​=maxfj​+w(i,j)

如果在 iii 增大时取到 fif_ifi​ 的最大值取值点 jjj 不降，我们称这个为具备决策单调性。

对于 w(i,j)w(i,j)w(i,j) 是否可以简单计算，我们对于这个问题存在若干种做法。

1 w(i,j)w(i, j)w(i,j) 可以简单计算：

我们可以直接单调栈维护可能的决策点，在压入时我们可以二分两者的管辖区间。

这样我们就有一个 1log 的简单做法。

2 w(i,j)w(i, j)w(i,j) 无法简单计算，但是可以简单维护加入删除。

2.1 dp 的转移不依赖前面的转移（离线）

我们可以做如下分治：

s(l,r,ol,or)s(l, r, ol, or)s(l,r,ol,or) 表示计算 [l,r][l, r][l,r] 值，决策点在 [ol,or][ol,or][ol,or] 的情况。

我们计算出 l,rl, rl,r 中点 midmidmid 的值与决策点，然后递归两侧即可。

时间复杂度是 1log 的，但是要乘上维护复杂度。

2.2 dp 的转移依赖前面的转移 （在线）

我们仍然做如下分治：

c(l,r)c(l, r)c(l,r) 表示计算 [l,r][l, r][l,r] 内的 dp 值。

我们先递归计算左半区间的 dp 值，也就是 c(l,mid)c(l, mid)c(l,mid)。

然后我们可以分治计算钦定了决策点在左半区间时右半区间的值，这个是离线的情况，也就是 s(mid+1,r,l,mid)s(mid + 1, r, l, mid)s(mid+1,r,l,mid)，可以分治解决。

最后我们再计算右半部分的情况。
