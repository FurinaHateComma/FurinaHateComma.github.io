0 - 约定

对于一个序列 aaa，其一般生成函数 G(a)G(a)G(a) 被认为是

g(x)=∑aixig(x) = \sum a_i x^i g(x)=∑ai​xi

这被称为生成函数的形式幂级数。

若 aaa 是一个无穷序列且有一定规律，那么 g(x)g(x)g(x) 一定可以视为对某个函数 f(x)f(x)f(x) 的逼近，我们令 F(a)=fF(a) = fF(a)=f 表示生成函数的封闭形式。

1 - 求解封闭形式

我们现在有一个无尽序列 aaa， aaa 一般以递推形式给出，如 fib\text{fib}fib，下面我们将以斐波那契数列为例子演示求解递推数列的生成函数封闭形式的方法。

思路是通过乘法消去绝大部分项。

首先我们有 fibi=fibi−1+fibi−2\text{fib}_i = \text{fib}_{i - 1} + \text{fib}_{i - 2}fibi​=fibi−1​+fibi−2​ 和 fib0=0\text{fib}_0 = 0fib0​=0， fib1=1\text{fib}_1 = 1fib1​=1。

我们写出这个东西的开放形式：

g(x)=∑fibixig(x) = \sum \text{fib}_i x^i g(x)=∑fibi​xi

观察这个东西的递推式，我们希望通过这个东西消除大部分项：

偏移可以通过乘上 xkx^kxk 来做到，具体的，我们有：

xk⋅g(x)=∑aixi+kx^k \cdot g(x) = \sum a_i x^{i + k} xk⋅g(x)=∑ai​xi+k

也就是

xk⋅g(x)=∑i=kai−kxix^k \cdot g(x) = \sum\limits_{i=k} a_{i-k} x^{i} xk⋅g(x)=i=k∑​ai−k​xi

我们把这个东西乘上 xxx 和 x2x^2x2 加起来就可以消掉大部分的项，余项是一次项上的 xxx。

g(x)−x2⋅g(x)−x⋅g(x)=xg(x) - x^2 \cdot g(x) - x \cdot g(x) = x g(x)−x2⋅g(x)−x⋅g(x)=x

进而有：

g(x)=x1−x−x2g(x) = \frac{x}{1 - x - x^2} g(x)=1−x−x2x​

2 - 求解形式幂级数

先明确目标，在一般生成函数中，我们有一个最基础的式子：

考虑如下序列的封闭形式：

a={1,r,r2,...rk,...}a = \{1, r, r^2, ... r^k, ... \} a={1,r,r2,...rk,...}

注意到有：

g(x)−rx⋅g(x)=1g(x) - r x \cdot g(x) = 1 g(x)−rx⋅g(x)=1

那么就是：

g(x)=11−rxg(x) = \frac{1}{1 - rx} g(x)=1−rx1​

我们的目标是把目标式子化成这个形式就可以转化回形式幂级数。

考虑一般形式的的目标式子形如：

g(x)=p(x)q(x)g(x) = \frac{p(x)}{q(x)} g(x)=q(x)p(x)​

钦定 p(x)p(x)p(x) 的最高次项低于 q(x)q(x)q(x)，否则我们可以提取高的部分出来。

接下来以斐波那契数列的封闭形式为例子：

g(x)=x1−x−x2g(x) = \frac{x}{1 - x - x^2} g(x)=1−x−x2x​

我们一定可以在复数域下将分母分解成若干个 1−rx1 - rx1−rx 因式子。

g(x)=x(1−r1x)(1−r2x)g(x) = \frac{x}{(1 - r_1 x) (1 - r_2 x)} g(x)=(1−r1​x)(1−r2​x)x​

这里 r1=−5−12，r2=5−12r_1 = \frac{-\sqrt{5} - 1}{2}， r_2 = \frac{\sqrt{5} - 1}{2}r1​=2−5​−1​，r2​=25​−1​。

接下来我们希望拆分成形如 A1−r1x+B1−r2x\frac{A}{1 - r_1x} + \frac{B}{1-r_2 x}1−r1​xA​+1−r2​xB​ 的形式

接下来给出一种方法：

P(x)(1−r1x)(1−r2x)…(1−rkx)=A11−r1x+A21−r2x+⋯+Ak1−rkx\frac{P(x)}{(1-r_1x)(1-r_2x)\dots(1-r_kx)} = \frac{A_1}{1-r_1x} + \frac{A_2}{1-r_2x} + \dots + \frac{A_k}{1-r_kx} (1−r1​x)(1−r2​x)…(1−rk​x)P(x)​=1−r1​xA1​​+1−r2​xA2​​+⋯+1−rk​xAk​​

我们对等式两边同时乘 (1−r1x)(1-r_1x)(1−r1​x)。

P(x)(1−r2x)(1−r3x)…(1−rkx)\frac{P(x)}{(1-r_2x)(1-r_3x)\dots(1-r_kx)} (1−r2​x)(1−r3​x)…(1−rk​x)P(x)​

等式右边：每一项都要乘以 (1−r1x)(1-r_1x)(1−r1​x)，结果如下：

A1+A2(1−r1x)1−r2x+A3(1−r1x)1−r3x+⋯+Ak(1−r1x)1−rkxA_1 + \frac{A_2(1-r_1x)}{1-r_2x} + \frac{A_3(1-r_1x)}{1-r_3x} + \dots + \frac{A_k(1-r_1x)}{1-r_kx} A1​+1−r2​xA2​(1−r1​x)​+1−r3​xA3​(1−r1​x)​+⋯+1−rk​xAk​(1−r1​x)​

令 x=1r1x = \frac{1}{r_1}x=r1​1​ （如果你觉得不能让 xxx 取等也可以取极限）

右边只剩下：A1A_1A1​。

左边变成了原本的分式，在“去掉了 (1−r1x)(1-r_1x)(1−r1​x)”之后，代入 x=1r1x = \frac{1}{r_1}x=r1​1​ 的结果：

P(x)(1−r2x)…(1−rkx)\frac{P(x)}{(1-r_2x)\dots(1-r_kx)} (1−r2​x)…(1−rk​x)P(x)​

A1=P(1r1)∏j=2k(1−rj⋅1r1)A_1 = \frac{P(\frac{1}{r_1})}{\prod_{j=2}^k (1 - r_j \cdot \frac{1}{r_1})} A1​=∏j=2k​(1−rj​⋅r1​1​)P(r1​1​)​
